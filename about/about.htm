TOC Extra

Descripción: :

extTOCExtra is a gvSIG extension originally developed by the Cartography Laboratory (Cartolab) from A Coruña University which adds new features on TOC contextual menu. Some of these new features are:

  • Change layer names on several layers at once.
  • Invert active layers.
  • Activate visible layers (only).
  • Show active layers (only).
  • Hide all layers.
  • Set the transparency at the same time for all the active raster layers
